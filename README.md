# HTML Final Project

This repository contains a **Pure HTML** final project with a landing page and six individual mini‑projects. Everything is built using only HTML (no external frameworks).

## Folder Structure

```
html-final-project/
|
|-- index.html          (Landing page)
|-- project1.html       (Multimedia Blog)
|-- project2.html       (Lists & Text Formatting)
|-- project3.html       (Anchor / Portfolio Page)
|-- project4.html       (Course Registration Form)
|-- project5.html       (Table Formatted Form)
|-- project6.html       (Travel Destination Guide)
|-- assets/             (Images, audio, video)
|-- resume.pdf          (Resume used in Project 3)
```

## What Each Page Does

- **index.html**
  Main landing page that links to all projects. It also includes a contact form and a map section.

- **project1.html – Multimedia Blog**
  Uses `audio` and `video` tags to play songs and videos.

- **project2.html – Lists & Text Formatting**
  Demonstrates ordered lists, unordered lists, description lists, and text formatting tags.

- **project3.html – Anchor / Portfolio Page**
  A simple portfolio using anchor links (`#about`, `#skills`, etc.) with social icons and resume download.

- **project4.html – Course Registration Form**
  A complete HTML form including inputs, radio buttons, checkboxes, and file upload.

- **project5.html – Table Formatted Form**
  A registration form laid out using an HTML table.

- **project6.html – Travel Destination Guide**
  Travel guide with anchor links and embedded Google Maps iframes.

## How to Run

### Option 1: Open Locally
Just open `index.html` in your browser.

### Option 2: Use Live Server
If you’re using VS Code:
1. Right‑click `index.html`
2. Click **Open with Live Server**

## Notes
- All assets are stored in the `assets/` folder.
- Project files use only HTML, no CSS frameworks or JS libraries.
